# Secondary Essay Rubric

Per-school secondary prompts (e.g., "why our school", diversity, adversity, leadership). Usually
short (often ~500–1500 chars). Reuse is fine, but each must answer THIS prompt for THIS school.

**Scored dimensions:** the shared dimensions from [`essay-base-rubric.md`](./essay-base-rubric.md)
that apply — **Specificity, Reflection Depth, Voice/Authenticity** — plus the overlay below. Score
each 1–10 with text-anchored reasoning.

> De-personalized: refer to "the applicant," never a name.

---

## Overlay — Prompt Fit / School Specificity

Does the essay actually answer the prompt, and (for "why us" prompts) is it specific to this school
rather than swappable?

| Score | Anchor |
|---|---|
| 1–2 | Doesn't answer the prompt, or generic "why us" that fits any school. |
| 3–4 | Loosely on-topic; school references are name-drops with no substance. |
| 5–6 | Answers the prompt; some genuine school-specific or prompt-specific detail. |
| 7–8 | Directly answers; cites concrete program/mission/community fit tied to the applicant's goals. |
| 9–10 | Could only have been written for this prompt + this school; fit is demonstrated, not asserted. |

---

## Scoring checklist
1. Score applicable base dims + Prompt Fit.
2. avg_score = mean of scored dims.
3. Write prose to `feedback/secondary-<school>.md`; track per-school status in the `schools[]` entry.
