# Content Directory

Copy this folder to `../content/` (gitignored) or set `CONTENT_DIR` to your existing vault folder.

Files the dashboard reads:
- Agent/scorecard.md — composite + domain scores + hard metrics
- Agent/competency-coverage.md — 17 AAMC competencies
- Agent/improvement-priorities.md — top improvement priorities
- Agent/meeting-todos.md — open/completed meeting todos
- Personal Statement/ps-scores.md
- Activities/activities-scores.md
- Impactful Experience/impactful-experience-scores.md
- Rec Letters/*.md — one file per recommender
- School List/Schools/*.md — one file per school
