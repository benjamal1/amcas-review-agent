# Meeting Notes

Place application-related meeting notes or transcripts here.

**Supported formats:**
- **Processed notes** (preferred) — structured markdown with action items, to-do lists, or checkboxes. The agent extracts directly from these.
- **Raw transcripts** — unstructured dialogue. The agent scans for action language and extracts implied tasks.

**Naming convention:** `YYYY-MM-DD [Meeting Name].md`
Example: `2026-04-01 AMCAS Advising - Ginger Fisher.md`

Once you've added a note, say: `"update meeting to-dos"`
